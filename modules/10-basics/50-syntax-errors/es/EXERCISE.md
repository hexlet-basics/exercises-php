El programa se ejecuta e informa del resultado. Escribe un programa que muestre:

```text
Programa iniciado correctamente
```

Una vez que el programa funcione, rómpelo a propósito: comete uno de los errores de sintaxis:

- quita el punto y coma `;`;
- quita la comilla o el paréntesis de cierre.

Ejecuta el código y lee el mensaje de PHP. Verás estos mensajes con frecuencia: es importante aprender a leerlos. Después, vuelve a la versión que funciona para que el ejercicio pase la comprobación.
