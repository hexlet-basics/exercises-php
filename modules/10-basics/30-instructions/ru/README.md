Команду `print_r('Hexlet')` мы можем назвать инструкцией, она говорит интерпретатору PHP что нужно делать. Таких инструкций может быть сколько угодно. Каждая из них выполняется после того, как закончилась предыдущая и так мы из простых элементов получаем сколь угодно большую и сложную программу.

```text
Инструкция 1: print_r('Hello')   →  выполнена
        ↓
Инструкция 2: print_r('World')   →  выполнена
        ↓
Инструкция 3: print_r('!')       →  выполнена
```

Код на PHP — это набор инструкций, которые отделяются друг от друга символом `;`. Вот пример кода с двумя инструкциями. Эти строки говорят компьютеру вывести фразы на экран.

```php
<?php

print_r('Mother of Dragons. '); // Первая инструкция
print_r('Dracarys!'); // Вторая инструкция
```

Результат выполнения:

```text
Mother of Dragons. Dracarys!
```

Обратите внимание: фразы вывелись друг за другом на одной строке. Команда `print_r()` выводит текст и ничего не добавляет от себя, поэтому перевод строки не появляется.

## Порядок имеет значение

Интерпретатор PHP выполняет код строго в том порядке, в котором вы его написали. Если поменять строки местами:

```php
<?php

print_r('Dracarys!');
print_r('Mother of Dragons. ');
```

на экране они тоже поменяются:

```text
Dracarys!Mother of Dragons. 
```

## Альтернативная форма записи

Обычно инструкции пишут на отдельных строках, но раз инструкции в PHP отделяются символом `;`, то перенос строки не обязателен. Несколько инструкций можно записать в одну строку:

```php
<?php

print_r('Mother of Dragons. '); print_r('Dracarys!');
```

Обе версии работают одинаково, но второй вариант читать сложнее. Поэтому инструкции почти всегда пишут по одной на строку.

## Зачем это нужно

Сейчас мы пишем очень простые программы, но со временем они начнут усложняться и один из самых важных навыков, который поможет их понимать, это способность разбивать (в голове) программу на независимые инструкции. Только так можно разобраться в том, что в коде происходит. Ниже пример для привлечения внимания, понимать его пока не надо:

```php
<?php

function isPrime(int $number): bool
{
    if ($number < 2) {
        return false;
    }

    $divider = 2;

    while ($divider <= $number / 2) {
        if ($number % $divider == 0) {
            return false;
        }

        $divider = $divider + 1;
    }

    return true;
}
```
