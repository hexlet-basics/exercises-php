El sitio genera automáticamente enlaces a las páginas de los usuarios, armándolos a partir de partes separadas. Arma el enlace usando la concatenación y muéstralo en la pantalla:

```text
https://github.com/hexlet/exercises-php
```

Cada componente de la URL es una cadena separada: el protocolo, el dominio, el nombre del propietario y el nombre del repositorio.
