Термометр показывает температуру с десятичной точкой. Программа должна вывести только целое значение с подписью.

У вас есть переменная:

```php
<?php

$temperature = 36.6;
```

Напишите программу, которая:

1. Преобразует температуру в целое число.
2. Полученное число превратит в строку.
3. Выведет эту строку на экран, добавив через пробел подпись `°C`.

Используйте операторы приведения типа `(int)` и `(string)`.
