A thermometer shows the temperature with a decimal point. The program should print only the integer value with a label.

You have a variable:

```php
<?php

$temperature = 36.6;
```

Write a program that:

1. Converts the temperature to an integer.
2. Turns the resulting number into a string.
3. Prints this string on the screen, adding the label `°C` after a space.

Use the type casting operators `(int)` and `(string)`.
