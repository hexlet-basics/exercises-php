Статус заказа обновляется по мере его продвижения. В начале заказ находится «в пути», затем переходит в статус «доставлен».

В упражнении определена переменная `$deliveryStatus` со значением `'в пути'`. Переопределите ее значение на `'доставлен'` и выведите на экран.

Пример переопределения переменной:

```php
<?php

$someVar = 'старое значение';
$someVar = 'новое значение';
print_r($someVar); // => новое значение
```
