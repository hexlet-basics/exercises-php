An order status is updated as the order progresses. At first the order is "in transit", then it moves to the status "delivered".

In the exercise, a variable `$deliveryStatus` is defined with the value `'в пути'`. Reassign its value to `'доставлен'` and print it to the screen.

Example of reassigning a variable:

```php
<?php

$someVar = 'old value';
$someVar = 'new value';
print_r($someVar); // => new value
```
