El estado de un pedido se actualiza a medida que avanza. Al principio el pedido está «en tránsito», luego pasa al estado «entregado».

En el ejercicio se define una variable `$deliveryStatus` con el valor `'в пути'`. Reasigna su valor a `'доставлен'` y muéstralo en pantalla.

Ejemplo de reasignación de una variable:

```php
<?php

$someVar = 'valor antiguo';
$someVar = 'valor nuevo';
print_r($someVar); // => valor nuevo
```
