Некоторые функции в PHP имеют **необязательные параметры**. Это значит, что для них заранее задано значение по умолчанию, и при вызове можно этот параметр не указывать.

![Автосервис](./assets/default-parameters.png)

Рассмотрим встроенную функцию `round()`, которая округляет число:

```php
<?php

$result = round(10.25, 0); // 10
```

Мы передали в нее два значения:

- Число, которое нужно округлить.
- Точность округления, `0` означает, что округление будет до целого числа.

Так как чаще всего нужно именно округление до целого, создатели функции `round()` сделали второй параметр необязательным и задали ему значение по умолчанию `0`. Поэтому результат будет тем же, даже если второй параметр не указывать:

```php
<?php

$result = round(10.25); // 10
```

Если нужна другая точность, то ее можно указать явно:

```php
<?php

// округление до одного знака после запятой
$result = round(10.25, 1); // 10.3
```

```text
round(10.25, 1)  →  аргументы: 10.25, 1    →  10.3
round(10.25)     →  аргументы: 10.25, (0)  →  10
                                       └── значение по умолчанию
```

Количество необязательных параметров зависит от конкретной функции, но обязательные всегда идут перед необязательными.

## Сигнатура функции

У каждой функции есть **сигнатура**, содержащая описание ее имени, параметров и порядка их использования. Сигнатура помогает понять, какие данные функция ожидает и что будет происходить, если параметры не указаны.

Посмотрим на [документацию](https://www.php.net/manual/ru/function.round.php) для функции `round()`. Если ее немного упростить, сигнатура выглядит так:

```php
round(int|float $num, int $precision = 0): float
```

Функция называется `round`. Параметр `$num` обязателен — он принимает число для округления (целое или дробное). Параметр `$precision` имеет значение по умолчанию `0`, то есть необязателен; если его не указать, округление произойдет до целого числа. После двоеточия указан тип результата — функция возвращает дробное число (`float`).

В полной сигнатуре из документации есть еще один необязательный параметр, который управляет режимом округления. Он нужен редко, поэтому мы его пока не рассматриваем.

## Как работать с новыми функциями

Когда вы встречаете новую функцию, можно использовать простой паттерн:

1. Открыть документацию и найти сигнатуру функции.
2. Посмотреть примеры использования.
3. Перейти в интерактивную оболочку PHP (она запускается командой `php -a`) и попробовать вызвать функцию с разными аргументами.

Такой подход помогает быстро разобраться, как именно работает функция, какие у нее обязательные и необязательные параметры и какие результаты она возвращает.
