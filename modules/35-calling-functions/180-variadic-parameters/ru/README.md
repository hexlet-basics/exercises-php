Некоторые функции обладают особенностью: они принимают переменное число аргументов. Это не связано со значениями по умолчанию, как в случае с `round()`. Речь идет о том, что количество передаваемых данных не ограничено фиксированным числом.

Рассмотрим функцию `max()`. Она находит наибольшее значение из переданных данных.

```php
<?php

print_r(max(1, 10, 3)); // => 10
print_r(max(1, -3, 2, 3, 2)); // => 3
```

Откроем [документацию](https://www.php.net/manual/ru/function.max.php) и посмотрим на сигнатуру `max()`:

```php
max(mixed $value, mixed ...$values): mixed
```

Это значит:

- функция требует как минимум одно значение (`$value`);
- дальше можно передавать еще сколько угодно значений (`...$values`) — переменное число аргументов обозначается многоточием `...`;
- функция вернет наибольшее из переданных.

Тип `mixed` в сигнатуре означает «любой тип»: функция умеет сравнивать не только числа, но и другие значения.

Если среди аргументов есть несколько одинаковых максимальных значений, возвращается первый из них.

```php
<?php

print_r(max(5, 5, 2)); // => 5
```

Аналогично работает функция `min()`, только она ищет наименьшее значение:

```php
<?php

print_r(min(1, 10, 3)); // => 1
print_r(min(1, -3, 2, 3, 2)); // => -3
```
