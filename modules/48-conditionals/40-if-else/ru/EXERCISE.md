
Реализуйте функцию `normalizeUrl()`, которая выполняет нормализацию данных. Она принимает адрес сайта и возвращает его с `https://` в начале.

Функция принимает адреса в виде `АДРЕС` или `http://АДРЕС`, но всегда возвращает адрес в виде `https://АДРЕС`. На вход функции также может поступить адрес в уже нормализованном виде `https://АДРЕС`, в этом случае ничего менять не надо.

Примеры вызова:

```php
<?php

normalizeUrl('https://ya.ru'); // 'https://ya.ru'
normalizeUrl('google.com');    // 'https://google.com'
normalizeUrl('http://ai.fi');  // 'https://ai.fi'
```

Есть несколько способов решить задачу. Один из них — проверить с помощью функции `str_starts_with()`, начинается ли строка-аргумент со строки `http://`, а потом на основе этого добавлять или не добавлять к ней `https://`.

Также вам скорее всего потребуется отбросить ненужную часть в начале строки. Помните, мы рассматривали способ получения кусочка от строки с помощью функции `substr()`? Если нет, напоминаю:

```php
<?php

// Берем 2 символа от начала
print_r(substr('python', 0, 2)); // => py
```

Так вот, с помощью `substr()` можно также отбросить определенное количество символов:

```php
<?php

// Отбрасываем первые 2 символа
print_r(substr('python', 2)); // => thon
```
