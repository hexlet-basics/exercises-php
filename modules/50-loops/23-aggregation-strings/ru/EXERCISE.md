Реализуйте функцию `sanitizePhoneNumber()`, которая принимает номер телефона из формы и возвращает строку без пробелов, скобок и дефисов.

Пользователи вводят номера по-разному, но перед сохранением их часто приводят к одному виду. Пройдите исходную строку посимвольно и соберите новый номер только из полезных символов.

```php
<?php

sanitizePhoneNumber('+7 (999) 123-45-67'); // '+79991234567'
sanitizePhoneNumber('8 800 555 35 35');    // '88005553535'
sanitizePhoneNumber('(123) 456-7890');     // '1234567890'
```

### Подсказки

* Итоговый результат — строка.
* Используйте пустую строку как начальное значение.
* Проверить, входит ли символ в набор `' ()-'`, поможет функция `str_contains()`.
