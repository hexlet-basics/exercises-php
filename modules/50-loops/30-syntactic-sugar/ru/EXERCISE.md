
Операторы синтаксического сахара особенно удобны, когда нужно постепенно собрать итоговое значение.

Реализуйте функцию `buildProgressBar()`, которая принимает количество выполненных шагов и общее количество шагов, а затем возвращает строку-индикатор прогресса.

Выполненные шаги обозначаются символом `#`, а оставшиеся — символом `-`. Старайтесь не использовать встроенные функции работы со строкой в своем решении.

```php
<?php

buildProgressBar(0, 5); // '-----'
buildProgressBar(3, 5); // '###--'
buildProgressBar(5, 5); // '#####'
```

Вам пригодится оператор `.=`, чтобы постепенно собирать новую строку внутри цикла `while`. А оператор `+=` будет полезен, чтобы управлять условием цикла.
