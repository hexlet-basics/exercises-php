Представьте, что у нас есть ряд чисел от 0 до 9. Мы хотим сложить эти числа. Мы могли бы сделать это так:

```php
<?php

$sum = 0;
$i = 0;

while ($i < 10) {
    $sum += $i;
    $i += 1;
}

print_r($sum); // => 45
```

Такой код мы можем переписать с помощью цикла `foreach` и функции `range()`:

```php
<?php

$sum = 0;

foreach (range(0, 9) as $i) {
    $sum += $i;
}

print_r($sum); // => 45
```

Функция `range()` создает последовательность чисел внутри определенного диапазона. Вызов `range(0, 9)` возвращает набор чисел от 0 до 9. А цикл `foreach` перебирает этот набор: на каждой итерации он записывает очередное число в переменную, указанную после ключевого слова `as`, и выполняет тело цикла.

Счетчик при этом не нужен. Цикл `foreach` сам знает, когда нужно остановиться — когда числа в наборе закончатся. Поэтому забыть изменить счетчик и получить бесконечный цикл здесь невозможно.

Функция `range()` поддерживает несколько вариантов вызова. Форма `range(start, end)` создает последовательность от `start` до `end`. Обратите внимание: в PHP конечное значение **входит** в диапазон — во многих других языках это не так. Форма `range(start, end, step)` создает последовательность с шагом `step`.

Наглядно варианты `range()` выглядят так:

```text
range(1, 5)     →  1, 2, 3, 4, 5
range(0, 3)     →  0, 1, 2, 3
range(0, 10, 2) →  0, 2, 4, 6, 8, 10
                    └── обе границы включены
```

Пример с суммой мы рассмотрели выше. Рассмотрим другой — распечатаем на экран числа от 1 до 3:

```php
<?php

foreach (range(1, 3) as $i) {
    print_r("{$i}\n");
}

// => 1
// => 2
// => 3
```

Теперь попробуем вывести числа в обратном порядке. Если начало диапазона больше конца, `range()` сам построит убывающую последовательность:

```php
<?php

foreach (range(3, 1) as $i) {
    print_r("{$i}\n");
}

// => 3
// => 2
// => 1
```

Конструкция `foreach` с функцией `range()` — самый короткий способ пройти по диапазону чисел. Когда же нужен полный контроль над счетчиком, например изменение по сложному правилу, на помощь приходят циклы `for` и `while`.
