С помощью цикла `while` решают любую задачу перебора элементов, но его отличает многословность. Для `while` нужно заранее вводить счетчик, задавать условие остановки и не забывать менять счетчик в теле цикла. Когда циклов немного, то это нормально, но в реальном коде циклы встречаются на каждом шагу. Поэтому управлять счетчиком вручную утомительно, особенно когда схема всегда одна и та же.

Для таких ситуаций в PHP есть цикл `for`. Он собирает все управление счетчиком в одном месте — в круглых скобках:

```text
while:                      for:

$i = 0;                     for ($i = 0; $i < 5; $i += 1) {
while ($i < 5) {                ...
    ...                     }
    $i += 1;
}
│                           │
└── управление разбросано   └── все управление в одной строке
```

Посмотрим, что находится в круглых скобках:

1. Начальное значение счетчика — этот код выполняется ровно один раз перед первой итерацией
2. Предикат (условие повторения цикла) — выполняется на каждой итерации, как и в `while`
3. Описание изменения счетчика — этот код выполняется в конце каждой итерации

В остальном принцип работы точно такой же, как у цикла `while`.

Посмотрим, как реализовать функцию переворота строки через цикл `for`. Такое задание встречается на собеседованиях:

```php
<?php

function reverseString(string $str): string
{
    // Начальное значение
    $result = '';

    for ($i = 0; $i < strlen($str); $i += 1) {
        $currentChar = $str[$i];
        // Соединяем в обратном порядке
        $result = "{$currentChar}{$result}";
    }

    // Цикл заканчивается, когда пройдена вся строка
    return $result;
}

print_r(reverseString('go!')); // => !og
```

Этот код можно прочитать так:

> Цикл с индексом `$i` повторяется, пока соблюдается условие `$i < strlen($str)`, и после каждого шага `$i` увеличивается на 1

Разберем подробно, как отрабатывает цикл в примере выше на каждом шаге итераций:

```text
reverseString('go!')

до цикла: result = ''

1 итерация: currentChar = 'g'
result = currentChar . result
       = 'g' . ''
       = 'g'

2 итерация: currentChar = 'o'
result = currentChar . result
       = 'o' . 'g'
       = 'og'

3 итерация: currentChar = '!'
result = currentChar . result
       = '!' . 'og'
       = '!og'
```

Теперь посчитаем количество упоминаний символа в строке без учета регистра:

```php
<?php

// $text - произвольный текст
// $char - символ, который нужно учитывать
function charsCount(string $text, string $char): int
{
    // Так как ищем сумму, то начальное значение 0
    $result = 0;

    for ($i = 0; $i < strlen($text); $i += 1) {
        // Приводим все к нижнему регистру,
        // чтобы не зависеть от текущего регистра
        if (strtolower($text[$i]) === strtolower($char)) {
            $result += 1;
        }
    }

    return $result;
}

charsCount('hexlet!', 'e'); // 2
charsCount('hExlet!', 'e'); // 2
charsCount('hExlet!', 'E'); // 2
charsCount('hexlet!', 'a'); // 0
```
