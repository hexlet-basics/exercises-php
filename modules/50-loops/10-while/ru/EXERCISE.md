Напишите функцию `printCountdown()`, которая имитирует обратный отсчет перед стартом. Функция принимает количество секунд и печатает числа от этого значения до `1`.

После завершения отсчета функция должна вывести на экран строку `Go!`.

```php
<?php

printCountdown(4);
// => 4
// => 3
// => 2
// => 1
// => Go!
```
