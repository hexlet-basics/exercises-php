Реализуйте функцию `generatePin()`. Она возвращает случайный четырехзначный PIN-код в виде строки.

Каждый разряд PIN-кода — независимое случайное целое число от 0 до 9.

```php
<?php

print_r(generatePin()); // => например, "4823" или "0571"
print_r(generatePin()); // => например, "1942" или "0037"
```

Укажите объявление типа возвращаемого значения.

## Подсказка

Сгенерируйте каждую из четырех цифр через `rand(0, 9)` и соберите строку через интерполяцию.
