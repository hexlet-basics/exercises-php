Реализуйте функцию `amountPerPerson()`. Она принимает сумму счета в ресторане, количество человек и процент чаевых, а возвращает сумму, которую платит каждый. Итог округляется вверх — никто не должен недоплачивать.

Для округления используйте встроенную функцию `ceil()`. Она доступна сразу — подключать ничего не нужно.

```php
<?php

// Счет 300, 4 человека, чаевые 20% — итого 360, каждый платит 90
print_r(amountPerPerson(300, 4, 20)); // => 90

// Счет 350, 3 человека, чаевые 10% — итого 385, каждый платит 129
print_r(amountPerPerson(350, 3, 10)); // => 129
```

## Подсказка

* Сначала посчитайте итоговую сумму с чаевыми, затем разделите на количество человек и округлите вверх
* `ceil()` возвращает число типа `float`, а функция должна вернуть `int` — пригодится приведение типов
