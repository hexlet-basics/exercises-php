When registering on a website, the program checks that the password is long enough — more than 5 characters. Write a function `isLongWord()` that returns `true` if the length of the given word is more than 5 characters, and `false` otherwise.

Example:

```php
<?php

var_dump(isLongWord('apple'));  // => bool(false)
var_dump(isLongWord('banana')); // => bool(true)
```
