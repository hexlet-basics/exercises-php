Напишите функцию `isFalsy()`, которая проверяет, трактуется ли переданное значение как ложное (falsy) с точки зрения PHP. Для выполнения этой проверки сравните переданное значение с `false`, используя нестрогое сравнение `==`.

```php
<?php

isFalsy(false);  // true
isFalsy(0);      // true
isFalsy('help'); // false
isFalsy(10);     // false
```
